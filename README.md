# 🚀 Taycan: OpenEnv Food Supply Chain Optimization Environment

This project simulates a real-world food supply chain where AI agents learn to optimize production, inventory, and reduce food waste.

## ⚙️ Environment Features

- Demand uncertainty simulation
- Inventory & expiry management
- Reward-based optimization
